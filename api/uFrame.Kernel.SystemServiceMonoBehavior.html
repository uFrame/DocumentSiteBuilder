Class SystemServiceMonoBehavior

The base class for all services on the kernel. Services provide an easy communication layer with the use of the EventAggregator. You can use this.Publish(new AnyType()). Or you can use this.OnEvent().Subscribe(anyTypeInstance=>{ }); In services you can also inject any instances that are setup in any of the SystemLoaders.
